# WordBlast 🚀

WordBlast is a premium, interactive word prediction puzzle game built with HTML, CSS, and Vanilla JavaScript. Decode the hidden word before you run out of lives!

## 🌟 Features

- **Modern UI/UX**: Sleek, dark mode aesthetics with glassmorphism components.
- **Dynamic Animations**: Satisfying hover effects, micro-interactions, smooth score scaling, and error shake animations.
- **Engaging Gameplay**: Attempt to guess the word symbol by symbol, or risk it all and guess the entire word.
- **Responsive Layout**: Adapts perfectly across screen sizes.

## 🕹️ How to Play

1. **Objective**: Guess the hidden five-letter word (`BLAST`).
2. **Guessing**: Type a single letter or the full 5-letter word in the prediction box and press `Enter` or click "Submit Guess".
3. **Scoring**:
   - Correct Letter = +20 points
   - Correct Word instantly = +100 points
4. **Lives**: You start with 3 lives (❤️). Each incorrect guess consumes one life. If you drop to 0 lives, it's Game Over!

## 🛠️ Stack

- **HTML5:** Semantic architecture.
- **CSS3 (Vanilla):** CSS features including Flexbox, animations, gradients, and custom properties (variables).
- **Vanilla JavaScript:** DOM Manipulation, event handling, logic, and state management. No external frameworks used.

## 🚀 Running Locally

There is no complex build process required! Since the application relies entirely on frontend technologies:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Enjoy the game!
